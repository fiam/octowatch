# Code of Conduct

This project aims to be straightforward, respectful, and useful.

## Expected Behavior

Participants in this repository are expected to:

- communicate respectfully and directly
- assume good intent, but stay precise about technical disagreements
- give feedback on code and decisions, not on people
- keep discussion focused on improving the project
- respect maintainers' time and project scope

## Unacceptable Behavior

The following are not acceptable:

- harassment, threats, or personal attacks
- discriminatory or demeaning language
- repeated bad-faith argument or deliberate disruption
- publishing private information without permission
- spam, trolling, or hostility that blocks productive collaboration

## Scope

This code of conduct applies to repository discussions, issues, pull
requests, and other project communication channels.

## Enforcement

Maintainers may edit, remove, or reject comments, issues, pull requests,
or contributions that do not follow this code of conduct.

Serious or repeated violations may result in temporary or permanent
removal from the project community.
